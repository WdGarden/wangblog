4. I hang out the clothes. → ________________________________________

## 🔄 Part 2: 连词成句

1. hat / I / on / head / my / wore / a  
   → ________________________________________  

2. gave / plant food / some / of / the plants / I  
   → ________________________________________  

3. the bird baths / with water / filled up / I  
   → ________________________________________  

4. the windows / washed / from the outside / I / all of  
   → ________________________________________  

5. school / ride / I / to / bus / the / catch / and  
   → ________________________________________  

6. math / before recess / have / we / and / English  
   → ________________________________________  

7. winter / build / we / in / snowmen / the  
   → ________________________________________  

8. my / help / do the dishes / mother / I / to  
   → ________________________________________  

<details>
<summary>查看答案</summary>

1. I wore a hat on my head.  
2. I gave some of the plants plant food.  
3. I filled up the bird baths with water.  
4. I washed all of the windows from the outside.  
5. I catch the school bus and ride to school.  
6. We have math and English before recess.  
7. In the winter we build snowmen.  
8. I help my mother to do the dishes.
