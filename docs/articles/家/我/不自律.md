
# 不自律


::: tip
一直很想写一些东西，记录自己的心路历程和琐碎的生活。如今已三十有四，生活仍是一地鸡毛。希望通过记录生活，不断反思总结，让生活逐渐步入正轨。
:::
1.**婚姻大事**还没有头绪，很是烦恼，真的想快点办完这件事。连随便找个人结婚都找不到，真的是自己的外在和性格影响太大了。其实，每个人一生中
都有几次结婚的机会，但每次都因为自己的胆怯或虚荣心而错过了。如今想来，真是后悔莫及。

2.自己的胆量很小，都已经是三十好几的人了，却总是很怯懦，不敢大胆地表达自己想法。一旦别人稍微反驳，就选择妥协，毫无主见。胆怯和无主见是相辅相成的。
关于胆怯的原因，我自己总结如下：第一，手淫容易让人产生自卑感。第二，很少和人沟通。想想很多事情都是相互影响的，最终导致了这样的性格和生活状态。

3.今天突然看到十年前办理社保卡时自己的照片。从这张照片的面相来看，很明显肾气不足，存在经常手淫的习惯。今天的我能看出来，想必当时的家人也能看出来，可是没有人劝诫我要节制、注意身体。也许他们也委婉提过吧，只是自己当时领悟不到。现在想想还是挺后悔的，如果早点戒掉这个不良习惯，我的人生应该会美好很多。
