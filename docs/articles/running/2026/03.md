

# RS232 vs RS485 速查

## 一、核心区别

| 项目 | RS232 | RS485 |
|------|-------|-------|
| 信号 | 单端（对地） | 差分（A-B） |
| 距离 | ≤15米 | 可达1200米 |
| 抗干扰 | 弱 | 强 |
| 线缆 | 普通多芯线 | **双绞屏蔽线** |
| DB9脚位 | 标准统一 | 不统一（常见38/12脚位） |

> **一句话**：232短距离一对一，485长距离抗干扰。111122333



## 二、线缆注意

- **232线**：多芯平行，不能用于485。
- **485线**：必须双绞+屏蔽，阻抗120Ω。

**结论**：设备是485，就用485线。



## 三、DB9脚位（485的坑）

| 方案 | A(+) | B(-) |
|------|------|------|
| 38脚位 | 针脚3 | 针脚8 |
| 12脚位 | 针脚1 | 针脚2 |

> 卡和线脚位不一致 → 直接插不通 → **改用绿色端子接线**，绕开DB9。



## 四、电子秤接线（485模式）

电子秤有两组端子：

| 模式 | 端子 |
|------|------|
| 485 | A / B / GND |
| 232 | TXO / RXO / PE |

**正确接法**（485模式）：
- 卡 A → 秤 A
- 卡 B → 秤 B
- 卡 GND → 秤 GND
- PE 不接（或只接屏蔽层）

> GND是信号地，PE是保护地，别混接。



## 五、你的实际接线

> **西霸485卡（绿色端子 A/B/GND） → 485专用线（剥开） → 电子秤（A/B/GND）**

- 不用转换器
- 不用DB9插头
- 10米内建议接GND
