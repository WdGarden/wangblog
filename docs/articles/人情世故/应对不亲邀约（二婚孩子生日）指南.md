# 应对不亲邀约（二婚/孩子生日）指南   

## 一、 问题描述
- **关系背景**：同门远亲，平时不走动，关系不亲。
- **邀约事由**：女儿二婚、外孙过生日等非核心红白事。
- **核心矛盾**：不去怕得罪人，去了觉得恶心、浪费钱，对方有利用人情敛财之嫌。

## 二、 核心心态
- **对方心理**：广撒网赌你脸皮薄，利用规则套取礼金。
- **我方原则**：**嘴上客气，行动坚决，钱包捂紧。**
- **目标**：不撕破脸的前提下，退出对方的“冤大头名单”。

## 三、 万能回应模板（首选策略）

### 适用场景：微信或电话接到邀请时

> **话术模板**：
> “哎呀真不巧，那天早就定好要出门（**带孩子比赛 / 单位封闭培训 / 去外地走亲戚**），实在赶不回来。恭喜恭喜啊，人不到心意到了，祝你们热闹开心！”

### 执行要点
1. **理由必须不在本地**：堵死对方“办完事晚点来”的后路。
2. **态度热情，祝福拉满**：伸手不打笑脸人。
3. **坚决不转账、不找人带礼**：只要不出钱，对方下次就不会叫你。

---

## 四、 被迫出席的保底方案

若因长辈压力等不可抗力必须到场，执行 **“最低能耗出席法”**：

| 环节         | 操作细节                                      |
| :----------- | :-------------------------------------------- |
| **到场时间** | 开席前 5-10 分钟到，吃完凉菜、新人敬完酒即撤  |
| **礼金金额** | 行情价打 4-5 折（例：行情 500 元，给 200 元） |
| **社交状态** | 吃完走人，不投入情绪，就当吃了顿快餐          |

---

## 五、 心态建设

- **不必愧疚**：对方明知关系远还叫，是对方先越界。
- **不必解释**：理由随口一说，不需要证明给谁看。
- **长期效果**：拒绝一次，下次名单就没你了。这是**帮你过滤无效社交**，是好事。

**记住：你的时间、金钱和好心情，只留给值得的人。**
